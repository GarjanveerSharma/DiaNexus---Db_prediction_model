## 📌 Overview

**DiaNexus** is a Machine Learning project that predicts whether a person is likely to have diabetes based on various medical attributes.

The project demonstrates the complete Machine Learning workflow—from data preprocessing and exploratory data analysis (EDA) to model training, evaluation, and prediction.

The primary objective is to build a reliable binary classification model that can assist in identifying potential diabetes cases using patient health data.

---

## 🚀 Features

- 📊 Data Cleaning & Preprocessing
- 📈 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning Model Training
- 📉 Model Evaluation
- 🎯 Diabetes Prediction
- 💾 Model Serialization
- 🧪 Prediction on New Data

---

## 🗂 Dataset

The project uses the **PIMA Indians Diabetes Dataset**, one of the most popular healthcare datasets for binary classification. It contains medical information such as:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Target)

The dataset is widely used for diabetes prediction research and educational machine learning projects. :contentReference[oaicite:0]{index=0}
